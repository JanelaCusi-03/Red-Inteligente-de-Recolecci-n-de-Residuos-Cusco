# Red Inteligente de Recolección de Residuos - Cusco

## Descripción
Este proyecto consiste en el desarrollo de un sistema web orientado a la gestión eficiente de pedidos y recolección de residuos en la ciudad de Cusco, integrando tecnologías modernas para optimizar el proceso.

## Objetivo
Facilitar la organización, seguimiento y control de pedidos mediante una plataforma web accesible.

## Tecnologías utilizadas
- Git (control de versiones)
- GitHub (repositorio remoto)
- HTML, CSS, JavaScript (base del sistema web)

## Instalación
1. Clonar el repositorio:
   git clone https://github.com/JanelaCusi-03/Red-Inteligente-de-Recoleccion-de-Residuos-Cusco.git

2. Abrir el proyecto en Visual Studio Code
